## Remote Control Project

### Week 1 Homework - Skill Distillery

#### Overview

This is a simulation of a simple remote control with buttons On, Off, and 0-9.  The remote responds with a text output upon press of a button.

#### Concepts

Do-while loops, simple methods, switch statement, scanners.

#### Technologies Used

Java

#### Lessons Learned

Manipulation of string variable to all caps, used to compare user input to string literal within Switch statement.  Case statement fall-through provides same treatment for multiple user responses.
